# Flutter zoom drawer App
## Setting Started

1.Dowload ZIP this Project or Clone Project 

2.Open VScode 

3.File >> Open Folder... (This project)

4.Click pubspec.yaml file 

5.Save file again  

6.Go to lib >> main.dart 

7.Select Emulator

8.Press F5 to run the program.

9.Finish
